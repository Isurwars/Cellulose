# Security Policy

## Supported Versions

Currently, the master branch and the latest release of Cellulose are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them to the maintainers privately via email at: **isurwars@gmail.com**.

You should receive a response within 48 hours. If the issue is confirmed as a vulnerability, we will open a private Security Advisory on GitHub, and discuss the timeline for a patch. Once the patch is available, we will publish the advisory and give you credit for the discovery.
